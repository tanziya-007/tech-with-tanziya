1. PROJECT OVERVIEW
================================================================

TechWithTanziya is a comprehensive learning platform designed to help students
and aspiring developers learn programming through structured content. The platform
features cheat sheets, blog articles, practical projects, learning roadmaps, and
downloadable resources.

Key Features:
- Responsive landing page with hero section
- Cheat sheet library with categorized content
- Blog section with programming articles
- Project showcase with technology stacks
- Learning roadmaps with step-by-step guides
- Resource downloads (templates, notes, etc.)
- Admin panel for content management
- Light/dark mode toggle
- SEO-optimized metadata

================================================================
2. TECHNOLOGY ARCHITECTURE
================================================================

2.1 Frontend Architecture:
- Framework: Next.js 15.2.0 (App Router)
- UI Library: React 18.3.1
- Language: TypeScript 5.5.4
- Styling: CSS Modules with custom properties
- State Management: React Context API
- API Communication: Fetch API with custom hooks

2.2 Backend Architecture:
- Runtime: Node.js with Express.js
- Database: MongoDB with Mongoose ODM
- Authentication: Google OAuth (for admin panel)
- File Storage: Google Drive API integration
- CORS: Cross-Origin Resource Sharing enabled

2.3 Development Tools:
- Package Manager: npm
- Linting: ESLint with Next.js configuration
- Build Tool: Next.js built-in
- Environment: TypeScript strict mode enabled

7. SETUP AND INSTALLATION INSTRUCTIONS
================================================================

7.1 Prerequisites:
- Node.js (v18 or higher)
- npm (v9 or higher)
- MongoDB Atlas account or local MongoDB installation
- Google Cloud Platform account (for Google Drive API)

7.2 Frontend Setup:
1. Clone the repository
2. Navigate to project directory: `cd tech-with-tanziya`
3. Install dependencies: `npm install`
4. Create `.env.local` file with required variables
5. Start development server: `npm run dev`
6. Open browser at `http://localhost:3000`

7.3 Backend Setup:
1. Navigate to backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Create `.env` file from `.env.example`
4. Configure MongoDB connection string
5. Set up Google Drive API credentials
6. Start backend server: `npm run dev`
7. Backend runs on `http://localhost:5000`

7.4 Environment Variables:
Frontend (.env.local):
- NEXT_PUBLIC_API_URL=http://localhost:5000
- NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id

Backend (.env):
- MONGODB_URI=your_mongodb_connection_string
- PORT=5000
- CORS_ORIGIN=http://localhost:3000
- GOOGLE_CLIENT_ID=your_google_client_id
- GOOGLE_CLIENT_SECRET=your_google_client_secret
- GOOGLE_REDIRECT_URI=http://localhost:5000/auth/google/callback
