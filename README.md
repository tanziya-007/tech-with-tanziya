# TechWithTanziya

A content-first Next.js website built for a growing learning platform brand.

## What is included

- Landing page with hero, cheat sheet previews, blog highlights, project showcase, roadmaps, and resources
- Cheat sheet category pages and individual sheet pages
- Blog, projects, resources, roadmaps, about, contact, and academy coming soon pages
- Light/dark mode UI with local preference persistence
- SEO-ready metadata and responsive layout

## Getting started

```bash
cd tech-with-tanziya
npm install
npm run dev
`

Then open `http://localhost:3000` in your browser.
